# Interactive Web Expense Tracker

A fully interactive, frontend web application built to help users seamlessly log daily expenses, categorize their spending, and track their total outgoings in real-time. 

🔗 **[View the Live Application Here](https://guy-gits.github.io/Web-Expense-Tracker/)**

## 🌟 Features
* **Real-Time DOM Manipulation:** Expenses instantly appear in the UI the moment they are logged, without requiring a page refresh.
* **Dynamic Calculations:** A running total automatically updates as items are added or removed from the list.
* **Persistent Local Storage:** Leverages the browser's native `localStorage` API to save data permanently, meaning your expense list survives browser restarts without needing a backend database.
* **Responsive Dark UI:** A sleek, modern aesthetic designed for readability and user comfort.

## 💻 Technologies Used
* **HTML5:** Semantic structure and form handling.
* **CSS3:** Custom styling, flexbox layouts, and interactive hover states.
* **JavaScript (ES6+):** Event listeners, array manipulation, and Local Storage integration.

## 🚀 How to Run Locally
Because this is a purely frontend application, no server setup is required.
1. Clone this repository to your local machine.
2. Open the folder and double-click the `index.html` file to run it in any modern web browser.
